# D017 监管后台端地图页面重构为地图工作台

## 背景

监管后台端地图页原型仍保留“筛选条 + 左侧图层面板 + 中间地图 + 全局抽屉承接详情”的旧结构。该结构与最新的地图工作台搭建清单不一致，也无法充分表达“顶部连续过滤 + 大地图 + 右侧详情承接 + 底部统计收口”的空间监管工作台体验。

## 本次变更

- 推翻旧地图页主体布局，不再采用左侧图层面板。
- 地图页改为页面内双栏工作台：左侧为筛选、图层、地图和底部统计；右侧为固定详情卡。
- 点位点击后在右侧详情卡内刷新对象摘要、事件状态和关联动作，不再默认展开全局抽屉。
- 底部统计条扩展为 8 项，与地图工作台 HTML 原型搭建清单保持一致。

## 采用方案

采用 `02_监管后台端地图工作台HTML原型搭建清单_v1.0.md` 中定义的结构：

```text
顶部标题栏
第一排筛选条
第二排图层开关条
左侧大地图
右侧事件详情卡
底部统计条
```

## 未采用方案

- 不继续微调旧版左侧图层面板，因为旧结构会压缩地图主视觉空间。
- 不把详情继续放在全局抽屉中，因为地图页需要在同一视野内完成点位查看和轻量闭环判断。
- 不新增独立事件中心入口，仍遵守监管后台端一期四页范围。
