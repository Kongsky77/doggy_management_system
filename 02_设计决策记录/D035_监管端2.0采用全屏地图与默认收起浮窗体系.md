# D035 监管端 2.0 采用全屏地图与默认收起浮窗体系

## 1. 背景

监管端 PRD v2.0 已确认单页地图产品形态。进入设计规范阶段后，进一步确认品牌色、宽桌面范围和右侧详情默认状态。

## 2. 最终决策

- 继续保留 CDHT 橙色品牌色
- 主要画布采用 `1920 × 1080`
- 兼容最小 `1440px` 宽桌面
- 不考虑移动端
- 地图铺满可用视口
- KPI、筛选、AI 摘要和右侧详情均作为地图上方浮窗
- 右侧详情首次进入默认收起
- 详情打开时覆盖地图，不压缩或重排地图容器

## 3. 采用原因

- 领导打开页面时可以首先看到最大范围的犬只分布
- 详情和 AI 只有在需要时展开，降低首屏信息压迫
- 覆盖式详情可以保持地图上下文和空间位置不变
- 浮窗体系比传统侧边栏和通栏驾驶舱更符合单页地图产品定位

## 4. 品牌与图层关系

品牌橙只用于品牌标识、主要操作和选中状态，不替代当前地图图层颜色：

- 犬只紫色
- 服务商黄色
- 宠物友好乐园绿色
- 犬只禁入区域红色

## 5. 影响范围

- `01_项目PRD及开发/02_监管后台端DESIGN_v2.1.md`
- 后续新建监管端 HTML 原型及独立资产
- 页面骨架、浮层尺寸、响应式、交互与验收
