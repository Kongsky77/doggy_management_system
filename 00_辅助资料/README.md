# 00 辅助资料

本目录用于存放帮助理解业务、方案和上下文的辅助性资料。

## 适合放入的内容

- 设计方案草稿
- 概念图、生态图、参考图片
- 交接稿、讨论纪要、思路整理
- 尚未成为正式需求依据的分析材料

## 不适合放入的内容

- 当前有效 PRD
- 当前有效原型 HTML
- 已确认并需要执行的正式需求说明
- 需求变更的正式决策记录

## 与其他目录的关系

- 如果某份辅助资料中的结论被正式采纳，应同步进入 `01_项目PRD及开发/` 或 `02_设计决策记录/`。
- 本目录资料可以作为背景输入，但不能直接替代 PRD、原型或决策记录。
- 历史辅助资料可以保留在本目录，但应尽量通过文件名或文档头部标明版本和状态。
